JMT-Services Project

All Projects JMT-Services is a contractor/job management tool I developed for fun. It’s built with a bunch of nifty technologies that you’ve probably heard of: Spring Boot, Spring Security, Google Cloud Platform (GCP), Thymeleaf, Keycloak, Bootstrap, and WebFlux. The idea was to create something that could handle the ups and downs of job management demands. With WebFlux, it’s as responsive as they come, scaling smoothly and swiftly. The whole thing runs in containers on GCP, making deployment a breeze, especially with a CI/CD pipeline managed through GitHub Actions.
